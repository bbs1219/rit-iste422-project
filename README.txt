How to run the EdgeConvert app from the jar file (EdgeConvert-1.0-SNAPSHOT.jar) 

1. Ensure your machine has Java installed and updated.
	If in doubt, update to the latest version.
	You can find the latest version at https://www.oracle.com/java/technologies/downloads/
	Download and follow the site's steps for installation at the following link:
	https://docs.oracle.com/en/java/javase/19/install/overview-jdk-installation.html
	

2. Open a terminal in whatever flavor your OS prefers
	Type java --version to confirm that you have Java working on your machine
	in your terminal, navigate to the directory containing EdgeConvert-1.0-SNAPSHOT.jar
	Once you're in the same directory as the file, type the following:

	java -jar EdgeConvert-1.0-SNAPSHOT.jar

	The app should launch, and from there you can use it as needed.
